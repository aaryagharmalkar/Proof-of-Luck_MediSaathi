# MediSaathi Mobile (React Native / Expo)

React Native app for MediSaathi, built with Expo.

## Setup

```bash
cd frontend/mobile
npm install
```

## Run

```bash
npm start
# or: npx expo start
```

Then scan the QR code with Expo Go (Android) or the Camera app (iOS).

## From repo root

```bash
make frontend-mobile
# or: make frontend-mobile-setup  # install deps first
```
